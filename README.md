Project Overview

This project focuses on building a Generative AI-based chatbot that can answer user queries from a medical document. The solution leverages transformer-based models from Hugging Face to understand and generate responses. The chatbot is deployed using Gradio, providing an interactive web interface for real-time querying.

🚀 Features
📄 Query-based interaction with medical documents
🤖 Uses pre-trained LLMs for intelligent responses
⚡ Fast and user-friendly interface
🌐 Deployed using Gradio for easy access
🔍 Context-aware answer generation
🛠️ Tech Stack
Python
Hugging Face Transformers
Gradio
NLP (Natural Language Processing)
⚙️ How It Works
Medical document is loaded and processed
Text is converted into embeddings / context
User inputs a query via the interface
The model retrieves relevant context and generates an answer
Response is displayed instantly on the UI
